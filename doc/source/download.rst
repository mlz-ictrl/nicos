Download
========

NICOS is available for download as a complete source file, using ``pip``, or
directly from the Git repository.

Source file
"""""""""""

The source distribution of NICOS can be downloaded from our
`download site <https://nicos-controls.org/download/>`_.

Pip
"""

Using pip NICOS can be installed::

    pip install -i http://forge.frm2.tum.de/simple nicos

where ``http://forge.frm2.tum.de/simple/`` is a FRM II local pip repository.


.. _clone-git:

Git repository
""""""""""""""

.. _nicos_repo:

The current development code can be obtained from our Git repository::

    git clone https://forge.frm2.tum.de/review/frm2/nicos/nicos

.. Non-users of `Git`_ can obtain the sources of the `latest development version`_.

.. _Git: https://git-scm.com


Developer access
""""""""""""""""

The access is granted in a two-step process:

 - First request access to `NICOS Gitlab <https://gitlab.com/nicos-controls>`_.

   Please, don't create an account via
   `the project page <https://forge.frm2.tum.de/redmine/project/nicos>`_,
   that would be limited to ticket system only.
 - You will find the account creation link in the
   `NICOS governance wiki (restricted access) <https://gitlab.com/nicos-controls/governance/-/wikis/home>`_.
 - If you are coming from a new institution, you may need to ask for your
   ip-range to get whitelisted.
